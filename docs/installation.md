

# 安装

要安装 TkEasyGo，你可以使用 `pip` 从 PyPI 安装，或者从 GitHub 克隆源代码并使用 `setup.py` 安装。

## 使用 `pip` 安装

```sh
pip install TkEasyGo
```

## 从 GitHub 克隆并安装

1. 克隆源代码：
   ```sh
   git clone git@github.com:TkEasyGo/TkEasyGo.git
   cd TkEasyGo
   ```

2. 使用 `setup.py` 安装：
   ```sh
   python setup.py install
   ```

   或者，你也可以使用 `pip` 安装：
   ```sh
   pip install .
   ```
